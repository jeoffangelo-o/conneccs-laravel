🔧 Cache Clear Helper

Use this page to help clear your browser cache and verify the fix is loaded.

Step 1: Stop & Restart Dev Server

In your terminal:

  1. Press Ctrl + C to stop the server
  2. Run: npx expo start --clear
  3. Wait for "Metro waiting on..." message

Step 2: Clear Browser Cache

⚠️ Important: Regular refresh (F5) won't work! You MUST hard refresh.

Method 1: DevTools (Recommended)

  1. Press F12 to open DevTools
  2. Right-click the refresh button (next to address bar)
  3. Select "Empty Cache and Hard Reload"

Method 2: Keyboard Shortcut

Press Ctrl + Shift + R (Windows) or Cmd + Shift + R (Mac)

Method 3: Clear All Cache

  1. Press Ctrl + Shift + Delete
  2. Select "Cached images and files"
  3. Select "All time"
  4. Click "Clear data"
  5. Then press Ctrl + Shift + R

Method 4: Use Incognito (Easiest!)

  1. Press Ctrl + Shift + N (Chrome) or Ctrl + Shift + P (Firefox)
  2. Go to http://localhost:8081
  3. This bypasses all cache!

Step 3: Verify Fix

✅ How to know it worked:
  • Open DevTools Console (F12)
  • Look for: 🔄 ReportorialFolderScreen loaded - Version 2.0 (Fixed)
  • Look for: Loading reportorial data - stored version: 2
  • Look for: Initial requirements loaded: 17
  • Navigate to Reportorial Requirements
  • Should see 15 requirement cards
  • Click any card - should open WITHOUT StyleSheet error

🆘 Still Not Working?

Nuclear Option:

  1. Close ALL browser tabs with localhost
  2. Stop dev server (Ctrl+C)
  3. Delete cache folders (see terminal commands below)
  4. Restart: npx expo start --clear
  5. Open in NEW Incognito window

Terminal commands:

rmdir /s /q node_modules\.cache
rmdir /s /q .expo
